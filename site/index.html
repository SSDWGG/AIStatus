What it watches

只做一件事：直视本机 AI 任务状态

01

菜单栏状态灯

蓝灯代表 GPT 或 Claude 正在处理任务，绿灯代表两者都空闲。无需切回终端就能扫一眼确认。

02

会话标题列表

读取 Codex 和 Claude Code 的本机会话事件，展示活跃和闲置会话标题，方便快速定位最近任务。

03

防休眠开关

需要长时间等待模型输出时，可手动开启保持活跃，阻止系统和显示器因空闲进入睡眠。

04

结束通知

当活跃会话变为空闲时，桌面通知会提示哪个会话刚结束，减少反复查看窗口的打断。

Local first

本地解析，不上传会话正文。

AiStatus 只读取运行状态和用于展示的会话标题。它不复制完整会话正文，也不需要远程账号。 当前版本面向本机使用场景，所有状态判断都在你的 Mac 上完成。

  • 读取 ~/.codex/sessions 的任务事件
  • 读取 ~/.claude/projects 的 Claude Code 事件
  • 防休眠功能必须由用户手动开启
