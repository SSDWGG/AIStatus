What it watches

只做一件事：直视本机 AI 任务状态

状态灯直接贴在菜单栏里，适合在多个窗口之间切换时保留一个持续可见的任务信号。

列表把 GPT 与 Claude 的状态放在同一处，减少分别打开终端、日志和编辑器确认进度的来回切换。

防休眠使用 macOS 电源断言，只有用户手动开启时才生效，适合长时间任务和外接显示器场景。

通知只在状态从运行切到空闲时出现，让你能继续做别的事，同时知道哪个 AI 任务刚刚结束。

Local first

本地解析，不上传会话正文。

AiStatus 只读取运行状态和用于展示的会话标题。它不复制完整会话正文，也不需要远程账号。 当前版本面向本机使用场景，所有状态判断都在你的 Mac 上完成。

  • 读取 ~/.codex/sessions 的任务事件
  • 读取 ~/.claude/projects 的 Claude Code 事件
  • 防休眠功能必须由用户手动开启

Gatekeeper

如果提示“Apple 无法验证”，这样打开。

当前 DMG 使用非 Apple 认证证书签名，首次打开时 macOS 可能拦截。确认你从本站下载后，可以在系统设置中手动允许。

  1. 看到“Apple 无法验证是否包含可能危害...”提示时，先点“完成”或关闭提示。
  2. 打开“系统设置” → “隐私与安全性”。
  3. 滚动到“安全性”，找到 AiStatus 被阻止的提示，点击“仍要打开”。
  4. 输入密码或使用 Touch ID，再点击“打开”。之后即可正常启动。
